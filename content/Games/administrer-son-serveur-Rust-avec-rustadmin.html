Comment administrer son serveur Rust avec Rustadmin

Publié le 2016-08-20 00:00

Télécharger et lancer RustAdmin

RustAdmin est totalement gratuit et est disponible sur son site officiel.

Rendez-vous sur le site et cliquez simplement sur Download:

Il s'agit d'une archive zip nous n'avons donc pas besoin d'installer le logiciel. Vous pouvez l'extraire dans un dossier ou sur une clé usb (pratique si vous voyagez beaucoup).

Ensuite pour lancer RustAdmin double-cliquez sur RustAdmin.exe :

Configuration de l'outil

En français s'il vous plait

L'application est traduite en français. Pour changer de langue il suffit de sélectionner langues dans le menu puis French :

Configurer une connexion avec votre serveur Rust

Cliquez sur l'onglet Configuration :

Vous aurez besoin de saisir :

  • IP: L'adresse IP de votre serveur (ex: 192.168.1.10)
  • Port serveur: Le port public de votre serveur. Par exemple : 192.303.12.1:29015
  • Rcon port: Le port RCON il est défini dans le fichier serverconfig.cfg de votre serveur c'est la ligne rcon.port.

    Pour ceux qui possèdent un serveur hébergé par nitroserv, par défaut il suffit d'ajouter +10 à votre port serveur. Dans ce cas cela nous donne donc 29015 + 10 = 29025

  • Mot de passe: Il est également défini dans le fichier serverconfig.cfg à la ligne rcon.password

Puis ensuite n'oubliez pas d'enregistrer votre configuration en cliquant sur sauvegarder :

Si vous retournez ensuite dans l'onglet console vous aurez alors accès à toutes les possibilités offertes par RuntAdmin ;)
